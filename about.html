About Claude Artifacts Hub

A gallery for interactive HTML artifacts created with Claude

What is Claude Artifacts Hub?

Claude Artifacts Hub is a web application built with Google Apps Script that showcases interactive HTML artifacts created with Claude AI. It provides an elegant gallery interface for browsing, viewing, and sharing these artifacts.

How to Use

  1. Browse the gallery on the home page
  2. Click on any artifact to view its details
  3. Use "Open Full Screen" to see the artifact in action
  4. Copy the code to use it in your own projects
  5. Share artifact links with others

Technology Stack

Backend: Google Apps Script (V8 Runtime)
Frontend: Vanilla JavaScript
Styling: Pure CSS with Custom Properties
Deployment: CLASP & Google Cloud
